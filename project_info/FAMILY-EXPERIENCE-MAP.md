# Family Experience Map -- Easter 2026

## Overview

This document maps the complete end-to-end experience for every participant in the Easter 2026 Egg Hunt. It covers from the moment Dad starts hiding eggs on Saturday night through the moment the kids rip open their Easter treasure from the shed on Sunday morning.

**Participants:**
- Girl (age 7) -- primary puzzle solver for reading/language tasks
- Boy (age 12, RCM Grade 8 piano) -- primary solver for math/music tasks
- Parent(s) -- facilitator, photographer, tech support
- Remote family -- Grandma Juliaa (Italy), Grandma Vera + Grandpa Tolia (Ukraine), Aunt Katiia/cousins (Ukraine)

**Location:** Family home + backyard, Winnipeg, Manitoba
**Date:** Sunday, April 5, 2026 (Easter Sunday)
**Duration:** 60 minutes (timer), plus physical setup/celebration time

## Stage 2: The Outdoor Egg Hunt (Sunday Morning, ~15 minutes)

### Experience Timeline

```
+-- Kids wake up, Easter morning excitement --+
|                                              |
|  "There are 20 eggs hidden in the yard!"    |
|  "Find them ALL before we go inside!"       |
|                                              |
+-- Kids run outside with baskets/bags --------+
|                                              |
|  Search yard, garden, porch, trees          |
|  Collect eggs one by one                    |
|  Parent counts along: "That's 14... 15..."  |
|                                              |
+-- All 20 found ----- back inside -----------+
```

### Parent Role During Hunt
- Announce the hunt and the count (20 eggs total)
- Provide warm/cold hints if kids are stuck finding the last few
- Take photos/video of the excitement
- Count eggs as they are found to confirm all 20 are collected
- DO NOT let kids open eggs outside -- "bring them ALL inside first!"

### Edge Cases
| Scenario | Response |
|----------|----------|
| Kids can only find 19 | Parent gives directional hint for the last one. All 20 must be found. |
| Egg found open (slip fell out) | Have spare printed slips as backup. |
| Rain/bad weather | Hide eggs inside the house instead -- under cushions, behind books, etc. |
| One child dominates finding | Encourage teamwork: "Let your sister find the next one!" |

## Stage 4: Phase 0 -- Egg Registration (~10-15 minutes)

### Experience by Persona

**Girl (age 7):**
- Reads the word riddles aloud: "Who is Mickey Mouse's clumsy best friend?"
- Scans the word slips on the table
- Taps the matching word chip in the app (e.g., GOOFY)
- Celebrates each flip animation (Easter illustrations)
- Attempts simpler number riddles (N1: 15+8, N9: 50-3)

**Boy (age 12):**
- Tackles the harder math riddles (N4: 8x7, N7: 17x5)
- Helps sister with riddles she is stuck on
- Gets excited by N10 (piano has 88 keys) -- connects to his music knowledge
- Keeps track of which slips have been used

**Parent:**
- Observes and enjoys
- Helps if kids are stuck (but riddles have built-in hints)
- Takes photos of the kids working together
- Watches the progress bar fill (14/20... 17/20... 20/20!)

### Emotional Arc
```
Excitement -> Focus -> Teamwork -> Satisfaction -> CONFETTI -> Anticipation
   (start)    (solving)  (helping)   (almost done)  (20/20!)   (what's next?)
```

### Transition Moment
When riddle 20/20 is solved:
1. Confetti burst fills the screen
2. Message: "All eggs registered! Now the REAL challenge begins..."
3. Code Board appears at top with empty lock slots
4. Phase 1 Assignment 1 loads

This is a key emotional pivot -- from "fun warm-up" to "serious puzzle adventure."

## Stage 5: Phases 1-4 -- The Main Challenge (~30-40 minutes)

### Phase 1: Digit Discovery (Assignments 1-3)

**Assignment 1 -- The Great Egg Sum:**
Kids physically look at their number slips and add them up. Girl helps with easier additions, boy handles the harder mental math. This connects physical egg slips to digital puzzles beautifully. Three-step digital root is a "whoa, cool!" math moment.

**Assignment 2 -- The Scale Explorer:**
Boy walks to the actual piano in the house and plays a C major scale. Counts the keys. This is the first physical-digital crossover in the main game. Girl watches and counts along.

**Assignment 3 -- Grandma Juliaa's Video:**
EMOTIONAL HIGH POINT. Kids see Grandma on screen, speaking to them. Girl examines the word slips and counts N's. Family connection across continents.

### Phase 2: Letter Discovery (Assignments 4-7)

**Assignment 4 -- The Ssssnake Poem:**
Girl reads the poem aloud to everyone. The word "shed" is hidden in plain sight (line 3) but kids won't notice yet. Gentle, literacy-focused puzzle.

**Assignment 5 -- The Alphabet Chain:**
Both kids count through the alphabet. References back to the piano number (8). First time kids see connections BETWEEN puzzles -- "Oh, it uses the answer from before!"

**Assignment 6 -- Grandma Vera & Grandpa Tolia's Video:**
SECOND EMOTIONAL HIGH POINT. Ukrainian language, subtitles. "Khristos Voskres!" -- kids learn the Easter greeting. Simple answer but deeply personal.

**Assignment 7 -- The Vlad Equation:**
Boy leads (Vlad is his half-brother). Counting letters in a family member's name. "D as in Daryna!" -- connects to their cousin.

### Phase 3: Assembly (Assignments 8-10)

**Assignment 8 -- The Semitone Puzzle:**
Boy returns to the piano. This is the hardest individual puzzle -- semitone counting requires real music theory knowledge. Girl can help count. Multi-step math builds to 18. "EIGHTEEN IS THE FIRST NUMBER!" -- huge Code Board reveal.

**Assignment 9 -- The Cousin Calculator:**
THIRD EMOTIONAL HIGH POINT. Video from Ukraine. Interactive table where kids spell out each cousin's name and count letters. Personal, collaborative, joyful. Reveals 38.

**Assignment 10 -- The Mirror Riddle:**
Girl solves this one -- it is a pattern recognition puzzle. "First and last are the same." Easy but satisfying. Reveals 18 again. ALL THREE LOCK NUMBERS NOW VISIBLE: 18-38-18.

### Phase 4: Unlock! (Assignments 11-12)

**Assignment 11 -- The Word Scramble:**
Both kids drag letter tiles S-H-E-D into order. Progressive hints guide them if stuck. The moment they spell SHED: "Wait... like the shed in our backyard?!" Connection clicks.

**Assignment 12 -- The Grand Finale:**
CLIMAX. Screen erupts with confetti. Both codes displayed in full:
- Combination: 18 - 38 - 18
- Word: SHED

"RUN TO THE SHED!"

## Stage 6: The Physical Unlock (~5 minutes)

### Experience Timeline

```
+-- Kids SPRINT to the shed ---------------------+
|                                                 |
|  Find the combination lock on the shed door    |
|  Dial: 18 ... 38 ... 18                       |
|  CLICK! Lock opens!                            |
|                                                 |
|  Find the word lock on the container inside    |
|  Rotate letters: S ... H ... E ... D           |
|  CLICK! Lock opens!                            |
|                                                 |
|  TREASURE! Chocolate, gifts, Easter surprises  |
|                                                 |
+-- Celebration! --------------------------------+
```

### Parent Role
- Run with the kids (or follow with camera)
- Help if combination lock is tricky to dial (these can be finicky)
- Capture the moment the locks open on video
- Celebrate with them

### Edge Cases
| Scenario | Response |
|----------|----------|
| Combination lock jams | Parent has the code and can help -- sometimes you need to pull the shackle while dialing |
| Kids forget the code | App still shows it on Assignment 12 screen -- go back inside and check |
| Weather prevents going to shed | Have a backup indoor location with the same locks |
| Lock was set wrong | Parent verified Saturday night (checklist item 3-4). If still wrong, parent opens it. |

## Risk Registry

| Risk | Severity | Mitigation |
|------|----------|-----------|
| Videos not recorded in time | RED | Fallback text mode is fully functional |
| Combination lock hard to open | YELLOW | Parent verified it Saturday night; parent can assist |
| Timer too short for the kids | YELLOW | Soft timeout lets them continue; 30min buffer built in |
| Girl gets bored during piano puzzles | YELLOW | She has supporting roles; her star moments are Assignments 4 and 10 |
| Boy finds early riddles too easy | YELLOW | Complexity ramps up; semitone puzzle is genuinely challenging |
| App crashes mid-game | YELLOW | localStorage persistence; resume on reload |
| Rain on Easter morning | GREEN | Hide eggs indoors instead |
| Kids peek at the code in config | GREEN | Config is bundled, not visible in UI; kids won't think to look |
